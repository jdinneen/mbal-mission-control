Method report · engine swarm

Model scale is not the lever

We re-trained the modern machine-learning toolkit against the hardest fair baseline — a site’s own strictly-backward rolling history — in California, then re-pointed the same engines at foreign coasts. This is the full, honest scoreboard: every gated engine ties or loses.

The verified claim. Across a 26-engine re-training in California and re-pointing the same engines at three fully-tested foreign systems — Irish shellfish toxin, and Irish and Hong Kong beach bacteria — no modern ML architecture (GNN, Mamba, DANN, neural-ODE/FNO, diffusion, NNGP, time-series foundation models) beat a simple strictly-backward rolling-memory baseline on the fair onset / forward slice.

1California — 26 engines, complete null

Each engine self-reported against a weak comparison (persistence, seasonal average, or a base-rate stand-in). Re-scored against a richer rolling-memory tree on the fair onset / forward-in-time slice, the “wins” disappear. Five engines that looked SUPPORTED were weak-baseline artifacts; the one that looked strongest (a graph neural net) was demolished by a hard three-attack check.

The GNN hard-check (the one that looked strongest). It beats a naive neighbour-memory baseline on the onset slice by +0.033 — but a plain gradient-boosted tree on the same neighbour lags beats the GNN by −0.18 AP (CI excludes 0), and on a train-past / test-future split the GNN (0.325) loses to neighbour-memory (0.632), the richer tree (0.822), and even plain persistence (0.560). Its apparent edge was temporally leaky, re-skinned memory.

The one genuine California lift is a modest combined-channel stack (a tree/π ensemble, not a neural architecture) on the onset slice. That is the exception that proves the rule: it is careful ensembling of simple channels, not model scale.

The US-lake cyano run completed but was not gated on this pass (its raw numbers are full-slice, not a fair result) — it is a null only via the separate California cyano run, so treat it as a caveat. The Gulf Karenia engine timed out and produced no gated result — we do not count it. We do not claim “four coasts.”

3How this was checked

Every engine was scored on the decision-relevant slice (toxic / bacterial onset, or a strictly forward-in-time split), against the strongest single backward rolling baseline — the best of a seasonal climatology, a rolling exceedance-rate, and a rolling prior-value ranker — never against persistence alone, a seasonal average, or a self-flattering “union” baseline. Margins were checked with station-clustered bootstrap confidence intervals and permutation / key-shuffle nulls; a default-refuted adversarial reviewer re-derived the headline results from the raw ledgers, and a separate fact-check went through every number before this page was published. Ranking metrics were scored on the raw model, because per-fold calibration on tiny folds silently sinks a real ranker.

What this rules out. It is not that we picked the wrong California data. Model cleverness loses to a site’s own recent history everywhere we have looked — so the lever is not a bigger or fancier model. The escapes that did work were a smarter question (predict the onset, not the level) and a new place (zero-shot transfer to an unmonitored coast).
