What Pi Is

The bacteria pi vector is a five-coordinate, label-free physics proxy. It maps every site-day into the same process-coordinate system: source loading, UV/thermal decay-residence, delivered UV dose, wind transport, and flushing. The point of the interlingua is that California, Ireland, Florida, Washington, or Australia can be represented on the same axes before a model sees labels.

Data provenance

StepMeaning
Row keyBacteria rows supply site, latitude, longitude, and sample date. Bacteria concentration and exceedance labels are not pi inputs.
Grid joinSites snap to a 0.5 degree NASA POWER daily weather cell.
Weather inputsDaily air temperature T, shortwave radiation S, precipitation P, and wind W. UV-B and SST variants exist but are not the core five used in the hybrid gate.
Coordinate calculationDirect arithmetic. There is no fitting, no label tuning, and no county-specific calibration in the coordinate itself.

Did we invent it?

The exact five-coordinate proxy set, normalizers, damping choices, and validation protocol are our engineered implementation. The mathematical style is inherited from dimensional analysis, Buckingham-pi nondimensional groups, Damkohler/Peclet process ratios, and Q10-style temperature response. The honest phrase is physics-motivated proxy pi coordinates, not first-principles hydrodynamics.

Core formula

S_star = S / 200
P_star = max(P, 0) / 5
W_star = max(W, 0) / 5
k_star = S_star * 2 ** ((T - 20) / 10)

pi_damkohler = k_star / (1 + W_star ** 2)
pi_loading   = P_star / (1 + W_star)
pi_uv_dose   = S_star / (1 + P_star)
pi_peclet    = W_star ** 2 / (1 + W_star)
pi_flushing  = W_star ** 2 / (1 + W_star ** 2)
CoordinateScientist-facing interpretation
pi_damkohlerUV/thermal decay-residence proxy: higher solar/temperature decay opportunity, damped by wind exchange.
pi_loadingRainfall loading proxy: same-day precipitation source pressure, damped by wind dilution/transport.
pi_uv_doseDelivered germicidal radiation proxy: shortwave input attenuated by precipitation/cloud/runoff proxy.
pi_pecletWind transport proxy: advective forcing versus mixing scale, inspired by Peclet-style reasoning.
pi_flushingBounded wind flushing proxy that saturates toward 1 under strong wind.

Overclaim Boundaries

Safe claim: "Label-free physics-proxy pi coordinates support zero-shot FIB ranking, and when combined with local memory they improve station-held-out California bacteria ranking. County-held-out calibration is not yet strong enough for deployable probability claims."

Missing physics that still matter: tide/tidal prism, real catchment discharge, turbidity/optical attenuation, bathymetry, wave mixing, sewer overflow mechanics, and residence volume. Those are the next scientific inputs to test before claiming a mechanistic coastal-water model.

Do not sayReason
"Pi is a full hydrodynamic solution."The formulas are engineered proxies using weather-scale variables, not catchment/tide/bathymetry/residence models.
"Pi replaces local memory."Memory beats unified pi AUC 0.7637 vs 0.6124 across 12 coasts when target history exists.
"Pi is a deployable advisory probability model."Zero-shot is ranking-only and county-held-out calibrated hybrid gate fails strict deployment.
"The bacteria result proves tensor/interlingua foundation modeling."Learned/transport interlingua variants did not beat raw pi; the winning pieces here are raw physics pi and HGBT hybridization.
"Pi added to the full California nowcast is a confirmed driver-null escape."The older escape ledger records it as WASH/null against the full FEATS+rain+discharge baseline.
