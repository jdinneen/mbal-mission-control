Clean-to-dirty onset is the strongest public-health reframe

The useful operational question is not just "which beaches are dirty now?" It is "which currently clean beach-days are most likely to flip into an exceedance?" On that onset slice, the frozen bacteria model beats station memory by a large margin.

Sources: data.json built 2026-06-26 07:11 Pacific; reports/bacteria_evidence_report_2026_06_26/index.html; Pi context from reports/pi_interlingua_explainer_2026_06_26/ and reports/pi_coordinate_technical_appendix_2026_06_26/.

1) Verdict

claim_clean_to_dirty_onset - PASS retrospective / prospective lockbox pending.

The onset slice contains 80,951 rows and 6,022 events, with a base rate of about 0.074. The model AP is 0.4177 versus 0.1990 for station memory and about 0.139 for AB411 rain. That is a +0.2187 AP margin and about a 2.1x ratio over station memory.

Held-out unit: beach-day rows in the same frozen temporal evaluation family: train <= 2019, calibrate 2020-2021, test >= 2022. The final public breakthrough label remains gated until the post-2026-06-18 prospective lockbox has rows and is scored.

3) What The Onset Slice Means

The source names the slice onset prev clean. It filters to rows where the site was previously clean and asks whether the next observed condition flips into a bacteria exceedance. This is the right framing for surprise risk: it removes many obvious persistent-dirty cases and tests whether the model can rank new trouble before station memory alone would.

9) Pi / Interlingua Context

Pi interlingua does not explain the onset win and should not be described as the deployed mechanism. It is a separate cold-start result: useful when there is no local memory, not strong enough to replace a mature local bacteria model.

QuestionAnswer
Does zero-shot help outside California?Yes, in a bounded cold-start sense. The pi+logistic screen has mean AP 0.1511 vs base 0.0909 across 12 held-out domains and beats its source-stratum null 12/12.
Does zero-shot beat target memory?No. Unified Pi LOBO AUC is 0.6124 vs target-domain station-memory AUC 0.7637; Pi beats memory on 0/12 coasts.
Can Pi and memory be combined to boost the California model?The combined tests did not justify a top finding. Pi groups added to the California nowcast were a wash or regression: pi_groups dAP about -0.0025 vs FEATS+rain+discharge, and unified 7-pi regressed about -0.0109.
How are coordinates calculated?They are deterministic label-free weather proxies: S*=S/200, P*=max(P,0)/5, W*=max(W,0)/5, k*=S* x 2^((T-20)/10); then source loading, Damkohler-like decay, UV dose, Peclet-like transport, and flushing coordinates.

11) So What

This is the decision-relevant bacteria finding for sampling triage: the model is especially useful when a site was clean before and staff need to decide where new risk is likely to emerge. The result is strong enough for retrospective scientific reporting and internal operational planning, but not for final prospective-lockbox breakthrough language yet.
