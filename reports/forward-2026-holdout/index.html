Frozen bacteria model passes the held-from-fitting 2026 season

The January 5, 2026 to March 5, 2026 season gives the strongest single-number forward-season read in the current evidence ledger: the frozen model beats station memory with AP 0.6335 versus 0.3682.

Sources: data.json built 2026-06-26 07:11 Pacific; reports/bacteria_evidence_report_2026_06_26/index.html; frozen headline and gate artifacts; Pi context from reports/pi_interlingua_explainer_2026_06_26/ and reports/pi_coordinate_technical_appendix_2026_06_26/.

1) Verdict

claim_forward_2026_holdout_pass - PASS for held-from-fitting forward-season evidence / GATED for pristine post-freeze lockbox.

The current finding reports 2,487 rows, 463 events, base rate 0.1862, model AP 0.6335, station-memory AP 0.3682, AP margin +0.2653, ROC-AUC 0.848, and ECE 0.023.

Scope discipline: this is held from fitting and calibration, but it is not a pristine never-scored post-freeze lockbox. The source state defines the prospective lockbox cutoff as 2026-06-18; the public data ended on 2026-03-05, so that post-freeze lockbox had zero rows.

10) Pi / Interlingua Relevance

Pi is useful context for transfer and cold-start questions, but it is not the reason the forward-2026 California model works. The forward result is a local calibrated bacteria nowcast with station history and environmental features.

11) So What

This finding is the forward-season sanity check: after model selection, the frozen model still wins on a later 2026 season. It should be shown as a strong held-from-fitting result, not overclaimed as a completed prospective post-freeze lockbox. The operational implication is that the model generalizes forward in time enough to justify continued prospective monitoring and lockbox scoring.
