Bacteria nowcast beats station-memory status quo

A calibrated bacteria nowcast beats the station-memory status quo on the 2022+ deployable California beach-bacteria test stratum. This page is the full focused report for the current Featured card, not a stale placeholder.

Sources: data.json built 2026-06-26 07:11 Pacific; reports/bacteria_evidence_report_2026_06_26/index.html generated 2026-06-26 17:24 UTC; Pi context from reports/pi_interlingua_explainer_2026_06_26/ and reports/pi_coordinate_technical_appendix_2026_06_26/.

1) Verdict

claim_bacteria_station_memory_supported - PASS retrospective / GATED prospective vs station memory, N=89,321 rows / 8,999 events.

Held-out unit: beach-day rows in the temporal 2022+ test era. CV / split: train <= 2019, calibration 2020-2021, test >= 2022; county holdout is used as a spatial stress check.

The current snapshot card reports model AP 0.4992 vs station-memory AP 0.2782, delta AP +0.221 with clustered-bootstrap CI [0.1926, 0.2564] and ECE 0.0171. The broader bacteria evidence report's selected operational benchmark reports the calibrated HGBT + harmonic-tide model at AP 0.510, AUROC 0.868, ECE 0.009 on the same deployable EXCLUDE_SAN_DIEGO family of evaluation.

7) What About Pi / Interlingua?

Pi is relevant background, but it is not this Featured operational finding. The Pi lane is a label-free cold-start ranking screen, not a calibrated advisory replacement for a mature local memory model.

QuestionAnswer
Does zero-shot Pi work anywhere?Yes as a ranking screen: 12-domain pi+logistic mean AP 0.1511 vs mean base 0.0909; mean lift 1.7524; source-stratum null beaten 12/12.
Does it beat local memory?No. Unified Pi LOBO AUC 0.6124 vs target-domain memory AUC 0.7637; Pi beats memory on 0/12 coasts.
How are Pi coordinates calculated?From exogenous daily weather only: S*=S/200, P*=max(P,0)/5, W*=max(W,0)/5, k*=S* x 2^((T-20)/10), then pi_damkohler, pi_loading, pi_uv_dose, pi_peclet, and pi_flushing. No bacteria labels, exceedance history, county labels, or target labels enter the coordinate calculation.
Did we invent it?We did not invent dimensional analysis, Damkohler, Peclet, Q10 temperature response, or FIB environmental controls. We engineered this exact five-coordinate proxy set and validated it as a cold-start bacteria ranking representation.
Should this be promoted over the nowcast?No. The deployed bacteria model is local calibrated nowcasting with memory, prior lab, rain, discharge, harmonic tide, geometry, and calibration window.

Beach-water-quality staff can use this model as a retrospective deployment candidate to rank high-risk beach-days better than station memory, AB411 rain rules, and Virtual-Beach-class MLR. The practical use is advisory triage and sampling prioritization, with San Diego and prospective lockbox scoring called out as blockers for broad deployment language.
