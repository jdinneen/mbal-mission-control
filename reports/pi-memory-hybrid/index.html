How The π Coordinates Are Calculated

The π vector is a five-coordinate, label-free physics proxy. For each site-day, it converts globally available weather into process coordinates for source loading, UV/thermal decay-residence, delivered UV dose, wind transport, and flushing.

Input provenance

StepWhat happens
Row keysBacteria monitoring rows provide site, latitude, longitude, and sample date. The bacteria concentration and exceedance label are not used as π inputs.
Weather cellsSites are snapped to a 0.5 degree grid cell for the NASA POWER daily weather grid.
Daily weatherNASA POWER provides daily air temperature, precipitation, wind, shortwave radiation, UV-B, and UV index. Shortwave and UV-B are converted from MJ/m^2/day to daily-mean W/m^2 where needed.
Optional SST variantsNOAA OISST can provide daily ocean temperature for A/B variants, but the core five used here are the weather-derived coordinates below.
FormulaOnce T, S, P, and W exist for a site-day, the coordinates are direct arithmetic. There is no fitting step in coordinate calculation.

Core formula

S_star = S / 200
P_star = max(P, 0) / 5
W_star = max(W, 0) / 5
k_star = S_star * 2 ** ((T - 20) / 10)

pi_damkohler = k_star / (1 + W_star ** 2)
pi_loading   = P_star / (1 + W_star)
pi_uv_dose   = S_star / (1 + P_star)
pi_peclet    = W_star ** 2 / (1 + W_star)
pi_flushing  = W_star ** 2 / (1 + W_star ** 2)

Math and history

CoordinateInterpretation
pi_damkohlerUV/thermal decay-residence proxy. It borrows the Damkohler-number idea: compare reaction or decay opportunity against transport/residence.
pi_loadingRainfall loading proxy. Precipitation is treated as same-day source pressure, damped by wind-driven dilution/transport.
pi_uv_doseDelivered germicidal radiation proxy. Shortwave radiation rises with clear-sky/solar input and is attenuated by precipitation as a cloud/turbidity/runoff proxy.
pi_pecletWind-driven transport proxy. It borrows the Peclet-number idea: advective transport versus mixing/diffusion scale.
pi_flushingBounded flushing proxy. It saturates toward 1 as wind forcing increases, preventing wind from dominating unboundedly.

Did we invent it? The exact five-coordinate proxy set, normalizers, damping choices, and validation protocol are our engineered implementation. The mathematical style is not invented here: it comes from dimensional analysis, Buckingham-pi style nondimensional groups, Damkohler/Peclet process ratios, and Q10-style temperature response. The honest wording is physics-motivated proxy pi coordinates, not a first-principles hydrodynamic solution.

Zero-Shot Interlingua Context

The coordinates are called an interlingua because the same axes mean the same thing in every geography. A model trained in one domain can score another domain zero-shot after both are mapped into this shared process-coordinate system.

QuestionAnswer
Does zero-shot help predict bacteria outside California?Yes for cold-start ranking. The prior 12-domain pi+logistic result scored mean AP 0.151 versus base 0.091, with 12/12 held-out domains beating the source-stratum null.
Is this a calibrated advisory model outside California?No. The zero-shot evidence supports ranking/triage when local labels are absent. It does not establish honest probabilities for a new jurisdiction without calibration.
Why not just use zero-shot instead of memory?When local history exists, memory is much stronger than π alone in California. The new finding is that π still adds residual value when combined with memory.
Is this the pi-tensor finding?No. The bacteria result here is a hand-built physics-π plus memory hybrid. The existing reports explicitly do not promote a tensor/foundation-model bacteria claim over the raw π control.
